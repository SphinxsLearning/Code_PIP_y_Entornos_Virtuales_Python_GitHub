# Game

For running the Game. Use this command:

```sh
python3 ./Game/game_piedra_papel_o_pijera.py
```
# Graficas
For running Graficas. Use these commands:
```sh
cd Graficas/
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
