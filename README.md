# Game

For running the Game. Use this command:

```sh
python3 ./Game/game_piedra_papel_o_pijera.py
```
# Graficas
For running Graficas. Use these commands:
```sh
cd Graficas/
```
```sh
python3 -m venv env
```
```sh
source env/bin/activate
```
```sh
pip3 install -r requirements.txt
```
```sh
python3 main.py
```
